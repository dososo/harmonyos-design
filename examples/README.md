| 示例 | 重点 | 关键规则 |

正式发布案例还必须补充：

- DevEco Studio / SDK 版本；
- 构建日志；
- 真机录屏；
- 性能；
- 无障碍；
- 慢后端和失败状态；
- 无 Skill 基线对照；
- 可执行原型或真实实现证据。
