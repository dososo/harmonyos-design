# 贡献者行为准则 · Code of Conduct

## 我们的承诺 / Our Pledge

为营造开放、友善的社区环境，我们承诺：无论年龄、身心障碍、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向，参与本项目和社区的每个人都不受骚扰。

We pledge to make participation in our project and community a harassment-free experience for everyone, regardless of age, disability, ethnicity, gender identity and expression, level of experience, nationality, appearance, race, religion, or sexual orientation.

## 行为标准 / Standards

有助于营造积极环境的行为 / Positive behavior:

- 使用友善、包容的语言 / Using welcoming and inclusive language;
- 尊重不同的观点和经验 / Respecting differing viewpoints and experiences;
- 优雅地接受建设性批评 / Gracefully accepting constructive criticism;
- 关注对社区最有利的事情 / Focusing on what is best for the community.

不可接受的行为 / Unacceptable behavior:

- 带有性暗示的言语或图像 / Sexualized language or imagery;
- 挑衅、侮辱/贬损的评论，人身或政治攻击 / Trolling, insulting comments, personal or political attacks;
- 公开或私下骚扰 / Public or private harassment;
- 未经许可发布他人隐私信息 / Publishing others' private information without permission.

## 执行 / Enforcement

项目维护者有权删除、编辑或拒绝不符合本准则的内容。违规行为可通过 GitHub Issues 报告，所有投诉都会被审查与处理。

Project maintainers may remove, edit, or reject contributions that do not align with this Code of Conduct. Violations may be reported via GitHub Issues and will be reviewed and addressed.
